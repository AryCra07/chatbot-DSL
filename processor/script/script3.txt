Variable
    $name Text <客户朋友>
    $balance Float 0.0
    $bill Float 0.0

State Welcome
    Speak <你好，> + $name + <！我是商城客服小A>
    Speak <输入“查询”查询你的余额与账单；> + <输入“存款”进行存款操作；> + <输入”咨询“咨询咱家产品>
    Case Contains <查询>
        Goto Query
    Case Contains <存款>
        Goto Deposit
    Case Contains <咨询>
        Goto AskGood
    Default
        Speak <不好意思没听清，可以再说一遍嘛？>
        Goto Hello

State Hello Verified
    Speak <输入“查询”查询你的余额与账单；> + <输入“存款”进行存款操作；> + <输入”咨询“咨询咱家产品>
    Case Contains <查询>
        Goto Query
    Case Contains <存款>
        Goto Deposit
    Case Contains <咨询>
        Goto AskGood
    Default
        Speak <不好意思没听清，可以再说一遍嘛？>
        Goto Hello

State Query Verified
    Speak <你好，> + $name + <！你的余额是> + $balance + <，你的账单是> + $bill
    Speak <查询结束，祝您生活愉快！>
    Default
        Goto Hello
    Timer 10
        Speak <亲，你还在吗>
    Timer 20
        Speak <这边先给您退出了>
        Goto Hello

State Deposit Verified
    Speak <你好，> + $name + <！请问你想存多少钱？请告诉我一个数字>
    Case Type Float
        Update $balance Add Input
        Speak <好的，已经给您的卡中存入> + Input + <！您的卡里现在有> + $balance + <元。>
    Default
        Speak <不好意思，您说的不太清楚，我们再来一遍好吗？>
        Goto Deposit
    Timer 20
        Speak <存款服务先给您结束了，祝您生活愉快！>
        Goto Hello

State AskGood Verified
    Speak <白菜>
    Default
        Goto Hello

State Goodbye Verified
    Speak <bye bye>
    Default
    Timer 0
        Goto Welcome
    Timer 20
        Goto Hello
