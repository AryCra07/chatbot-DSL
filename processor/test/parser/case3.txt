Variable
    $name Text <Ayu>
    $billing Float 1.1
    $balance Float 10.0

State Welcome
    Speak <Hello, ayu>
    Default
        Goto Hello

State Hello Verified
    Speak <test1: > + <Hello, > + $name + <! Your balance is > + $balance + <.>
    Speak <Type 'ask' to ask me questions>
    Case Contains <ask>
        Goto Answer
    Case Length >= 50
        Speak <Your question is too long, ask again>
        Goto Hello
    Default
        Exit
    Timer 0
    Timer 500
        Update $transactions Add 1

Variable
    $transactions Int 0

State Goodbye Verified
    Speak <bye bye>
    Case Contains <goodbye>
        Update $billing Set 0
    Case <no>
        Speak $name + <timeout> + Input
        Update $billing Sub -1
        Update $name Set Input
    Case Type Int
        Speak <hh>
        Update $transactions Add Input
        Update $billing Add Input
    Case Type Float
        Update $billing Sub Input
    Default
    Timer 0
        Goto Welcome
    Timer 20
        Goto Hello

State Answer Verified
    Speak <Here's question>
    Case Contains <退货>
        Speak <不行捏>
        Update $billing Set 0
    Case <优惠>
        Speak <购买满500打八折>
    Case Type Int
        Speak <hh>
        Update $transactions Add Input
        Update $billing Add Input
    Case Type Float
        Update $billing Sub Input
    Default
    Timer 0
        Goto Welcome
    Timer 20
        Goto Hello
